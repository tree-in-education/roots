Facial Landmark Estimation

After the algorithm detects a face, it identifies facial features. Facial features include the mouth, eyes, and nose. Once it finds these features, it rotates and scales the image so that the face is centered.

Facial Encoding

Now that the algorithm has the facial features, it needs a way to find a way to identify a person from them. It needs to find what’s unique about each person’s face. So the computer encodes your face into a set of numbers. These numbers represent things like the distance between your nose, mouth, and eyes. To the algorithm, you are nothing but numbers.
