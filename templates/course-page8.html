Machine Learning

The algorithms for both face landmark detection and face encoding are created using a process called machine learning. Machine learning algorithms are algorithms that improve automatically through experience. The algorithm goes through a process, known as training, which helps it improve itself to learn what it is supposed to do. In the case of face encoding, the algorithm is supposed to learn what makes pictures of the same person similar, and what makes pictures of different faces look different.

Training Face Encoding

In training, the algorithm will be given two images of the same person, and one image of a different person. The algorithm quantifies what is similar between the two images of the same person, and what is different about the image of the different person. It may do this thousands of times, on thousands of images. Eventually, the algorithm generalizes, and learns to encode faces so that it encodes two images of the same face in a similar way.
