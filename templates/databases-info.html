Part 2: Intro to Databases

The most popular form of database is a relational database, which is one in which data is organized into one or more tables, with each table being related to another in some way.

Other types of databases include:

  • Centralised database
  • Distributed database
  • Personal database
  • End-user database
  • Commercial database
  • NoSQL database
  • Operational database
  • Cloud database
  • Object-oriented database
  • Graph database
