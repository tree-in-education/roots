Part 2: Intro to Databases

Like we said before, relational databases are one in which data is organized into one or more tables, with each table being related to another in some way.

Each table has a set of fields, which define the nature of the data stored in the table. A record is one instance of a set of fields in a table. To visualize this, think of the records as the rows of the table and the fields as the columns of the table. See the figure below.

In the example above, we have a table of student information, with each row representing a student and each column representing one piece of information about the student. Each row, or record, represents a student, and each column represents one piece of information about the student: their First Name, Last Name, Major, or Birthdate.
