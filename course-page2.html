How It Works

Facial recognition systems use algorithms to identify a person using their face. But what is an algorithm, really? An algorithm is simply a step by step process a computer follows. Most facial recognition algorithms happen in 4 steps:

  1. Face Detection: The computer finds each face in an image.
  2. Feature Detection: The computer finds facial features, like the eyes, mouth and nose.
  3. Face Encoding: Measurements between facial features are found.
  4. Face Comparison: The face is compared to faces in a database.
